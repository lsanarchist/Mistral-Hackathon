Plugin Information

Plugin Architecture

TriageProf uses a modular plugin system where each profiler lives in its own plugin module. Plugins are discovered via JSON manifests and communicate using JSON-RPC.

Plugin Locations

Plugins are located in the plugins/ directory with manifests in plugins/manifests/. Each plugin has its own source code and binary.

Extensible Design

New profilers can be added by creating plugins without modifying the core system. The plugin API remains stable for backward compatibility.
